# HealthBridge Frontend
This is the frontend of the HealthBridge application, a healthcare platform designed to connect users with medical services through a simple and responsive web interface.

## Overview
The frontend is built using React and communicates with backend APIs to fetch and display dynamic data. It follows a component-based architecture and provides a smooth user experience as a single-page application.

## Tech Stack
- React.js
- Vite
- Tailwind CSS
- Axios
- Vercel (Deployment)

## Features
- Responsive user interface
- Fast loading using Vite
- API integration using Axios
- Component-based UI structure
- Single Page Application (SPA)
- 
## Project Structure
- `src/` → Main application code (components, pages, logic)
- `public/` → Static assets
- `index.html` → Entry point
- `vite.config.js` → Vite configuration
- `tailwind.config.js` → Tailwind setup

## How It Works
1. User interacts with the UI
2. Frontend sends API requests using Axios
3. Backend responds with data
4. React updates the UI dynamically
